# Chancellerite Telegram bot

Installation

```
cd tgbot/code
pip install -r requirements.txt
```
On mac:
```
pip3 install virtualenv
virtualenv -p python3 <tgbotpath>
source <tgbotpath>/bin/activate
```

debian:
```
apt install python3-venv
```
or
```
pip3 install virtualenv
```

then
```
cd <tgbotpath>
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
nano .env
```
